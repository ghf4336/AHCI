About NapApp

NapApp is an android mobile application built by AUT students for the company LifeSpan Trust. The purpose of the application is to allow users to take a timed nap based on when they start to fall asleep. When the user falls asleep a timer is set to the amount of time they want to sleep for and an alarm will ring at the end to wake them up. NapApp is currently in development

Sleep Detection

Sleep detection is the method in which the app regonises that the user has fallen asleep. NapApp uses a tap detection method as its sleep detection. When the participants are trying fall asleep. The phone will vibrate every 10 seconds. Each vibration requires the user to to tap on the screen to notify the app that they are still awake. when the user misses 3 taps the timer for the nap starts, hence they fallen asleep.

Hypothesis

Is a vibration with a higher intensity is more likely to keep the user awake when using the app?
We believe the higher the vibration intensity the more likly the user will stay awake. This statemade is made based on past expereice we have had with the app. we hope to prove or disprove this hypothesis based on the test that we do.

Usability Test Summary

This usaiblity test will be done by Adrian Liow from Auckland University of Technology. This test will be done as an assignment for the Bachelor of Computer and Information Science paper, Applied Human Computer Interaction (COMP719) paper. The usablity test will be testing the hypothessis stated above with 6 participants.

Each participant will have to take the test individually and independent over the course of one week. They are expcted to comeplte a survey everytime they take a nap, which will be given to them when they get the app installed onto their phone.

Based on the results given from each participant, an analysis will be done to come up with some findings for the hypothesis. The findings will be made avaliable to the appropriate people
